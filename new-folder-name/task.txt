
Task: Create a simple CRUD API using Node.js and MongoDB.
Details: Implement the following routes:
POST /api/resource: Create a new resource.
GET /api/resource: Get a list of resources.
GET /api/resource/:id: Get a specific resource by its ID.
PUT /api/resource/:id: Update a resource by its ID.
DELETE /api/resource/:id: Delete a resource by its ID.
